Align

Texto alineado al centro

Line Height

Texto con mayor altura de línea para que el espacio entre renglones sea claro.

Decoration

Texto con decoración subrayada.

Transform

Texto transformado a mayúsculas.

Indent

Texto con sangría en la primera línea para simular un párrafo.

Spacing

Texto con espacio entre letras.

34 - Variables

The var() function is used to insert the value of a CSS variable.

Primary Color

This card uses the --primary-color CSS variable.

Secondary Color

Variables allow you to change colors globally from a single place.

Spacing Variable

Padding and margins can be controlled with reusable variables: --spacing: 1rem

Border Radius

The rounded corners use the --card-radius: 12px variable.

Theme Management

CSS variables make it easy to create light and dark themes. Try changing your OS theme!
